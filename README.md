# Sync Application

A comprehensive desktop application for managing media, campaigns, and operations with a classic 2020 UI design.

## Features

- **Multi-Window Support**: Open multiple windows simultaneously
- **Media Management**: Add, view, and manage media locations with comprehensive details
- **Campaign Management**: Create and manage bookings with advanced features
- **Expense Tracking**: Track and manage expenses with work order generation
- **PO Management**: Monitor and manage pending purchase orders
- **Classic 2020 UI**: Modern yet classic design aesthetic

## Modules

1. **File** - File operations
2. **Reception** - Reception management
3. **Media** - Media location management
4. **Sales** - Sales tracking
5. **Campaigns** - Campaign and booking management
6. **Operations** - Operations management
7. **Reports** - Reporting module
8. **Accounts** - Account management
9. **Masters** - Master data management
10. **Administration** - Administration settings

## Technology Stack

- Frontend Framework: React/WPF with Classic 2020 UI
- Multi-window capability for desktop application
- Real-time data synchronization
